# Website Đặt Đồ Ăn Nhẹ A2-Coffee-HaUI

## 📌 Chủ đề
Xây dựng website đặt đồ ăn nhẹ A2-Coffee-HaUI cho phép người dùng:
- Đăng ký, đăng nhập.
- Xem danh mục và sản phẩm.
- Thêm món ăn vào giỏ hàng.
- Đặt hàng và xem lịch sử đơn hàng.
